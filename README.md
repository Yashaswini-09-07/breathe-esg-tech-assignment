# Breathe ESG Tech Assignment

A prototype ESG Data Ingestion Platform built using Django and Django REST Framework.

## Features

- Organization Management
- Data Source Tracking
- Import Batch Tracking
- Activity Records API
- Audit Logging
- Sample ESG CSV Files

## Technology Stack

- Python 3.11
- Django 5
- Django REST Framework
- SQLite

## API Endpoint

GET /api/records/
